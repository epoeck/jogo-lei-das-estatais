# Jogo Lei das Estatais

Este projeto é um quiz sobre a Lei das Estatais para ser executado inteiramente no navegador.

## Como executar localmente

1. Clone o repositório e abra a pasta no terminal.
2. Rode um servidor estático, por exemplo:

```bash
python3 -m http.server
```

3. Acesse `http://localhost:8000` no navegador.

## Como publicar no GitHub Pages

1. Faça login no GitHub e crie um repositório para o jogo.
2. Envie os arquivos deste projeto com `git push`.
3. Na aba **Settings** do repositório, ative **GitHub Pages** escolhendo a branch desejada.
4. A página ficará disponível na URL informada pelo GitHub.
